Description

Bonjour is a common French greeting that can be used throughout the day, but it's traditionally associated with the morning. The word Bonjour is derived from the phrase "bon jour," which means "good day." In France, it's customary to use Bonjour when greeting someone in a formal or informal setting, regardless of the time of day.

If you're in France or a French-speaking country and want to greet someone in the evening, you can use Bonsoir instead of Bonjour.
