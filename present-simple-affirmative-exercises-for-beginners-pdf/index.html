Description

The Present Simple Affirmative Tense is used to describe actions that are habitual or routine. It is also used to describe facts that are generally true, as well as things that are always true. This tense is formed by using the base form of the verb (also known as the infinitive) with the third person singular (he, she, it). For example:

  • He plays soccer every Sunday.
  • She studies English every day.
  • It rains a lot in Seattle.

Using Present Simple Affirmative Tense correctly is important for effective communication in English. It is also a fundamental building block for more complex tenses, such as the Present Continuous and the Present Perfect.

How to Use Present Simple Affirmative Tense

To form a sentence in Present Simple Affirmative Tense, you need to use the base form of the verb with the third person singular (he, she, it). Here is the basic structure:

Subject + Verb (base form) + -s/-es + Object

For example:

  • She plays the guitar.
  • He walks to work every day.
  • It gets cold in the winter.

Note that when the verb ends in -s, -x, -z, -ch, or -sh, you need to add -es instead of -s. For example:

  • She watches TV every night. (Watch ends in -ch, so we add -es)
  • He fixes cars for a living. (Fix ends in -x, so we add -es)
